1. Function that checks if a given number is positive, negative, or zero.

Input: 0 = Output = "Zero"
Input: 5 = Output = "Positive"
Input: -5 = Output = "Negative"
Input: 999999 = Output = "Positive"
Input: -999999  = Output = "Negative"
Input: 0.0001 = Output = "Positive"
Input: -0.0001 = Output = "Negative"
Input: "abc" = Output = Error / Entrada invalida
